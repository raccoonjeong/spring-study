# 1. 함수 선언식이란?

자바스크립트에서 가장 기본적인 함수 정의 방식입니다.

```js
// 함수 선언식
function add(a, b) {
  return a + b;
}

console.log(add(2, 3)); // 5
```

이름 있는 함수를 직접 선언.

코드 어디에서든 호출 가능(호이스팅 적용됨).

리액트 컴포넌트를 만들 때도 자주 사용.

```jsx
// 리액트 함수 컴포넌트 - 선언식
function Hello() {
  return <h1>Hello World!</h1>;
}

export default Hello;
```
