# 2. 함수 표현식

## 함수 표현식이란?

변수에 함수를 할당하는 방식입니다. 익명 함수가 주로 사용됩니다.

```js
// 함수 표현식
const add = function (a, b) {
  return a + b;
};

console.log(add(2, 3)); // 5
```

변수에 함수를 담아 쓰는 방식.

선언식과 달리, 선언 이전에 호출 불가능(호이스팅 X).

## 리액트에서 사용법

리액트에서 화살표 함수와 함께 많이 사용됨.

```jsx
// 리액트 함수 컴포넌트 - 표현식
const Hello = function () {
  return <h1>Hello World!</h1>;
};

export default Hello;
```

### 화살표 함수 표현식

리액트에서 가장 많이 쓰는 형태입니다. 더 간결하고, this 바인딩 문제가 없음.

```js
// 화살표 함수 표현식
const add = (a, b) => a + b;

console.log(add(2, 3)); // 5
```

리액트 컴포넌트 예제:

```jsx
// 리액트 함수 컴포넌트 - 화살표 함수
const Hello = () => {
  return <h1>Hello World!</h1>;
};

export default Hello;
```
